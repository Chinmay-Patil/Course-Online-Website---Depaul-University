Introduction

The Course Online Remote Administration website is a web based remote administration tool for managing and monitoring the hardware and software equipment in each COL-enabled classroom, administering lecture recordings and carrying out various recording related functions. RA website consists of 5 sections Monitor, Schedule, Recording, Message, Log.

Monitor section is the default one which is used to perform monitoring of the all the ongoing recording, also it displays the real-time status of the recording equipment

Schedule section is used to view future recording schedule and to add new recordings to the system

Recording section is used to playback recorded lectures of various types.

Message section is an interactive page where instructor can send messages to the COL admin in case of an issue in any of the COL equipped classes. Also serves as internal chat for currently logged in COL admins

Log section displays the upload status of previously recorded lectures.
