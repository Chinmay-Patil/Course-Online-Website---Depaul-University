  • Tip 1: Try to start monitoring RA 10-15 minutes before classes begin so that any potential issues can be discovered and fixed beforehand
  • Tip 2: Run the “Group Schedules” command before the recording start time, to verify that upcoming recordings are present in the automatic schedule
  • Tip 3: Open each classroom group in a separate IE window and arrange the windows across both monitors
  • ❖ Steps to follow when monitoring recordings on RA:
    • ► When the automatic class recordings start, verify:
      • Room, Course Id, Duration for each class.
    • ► Verify that the Special Lab recordings for that time slot have started, if any
    • ► Start TEST recordings for any other observed activity
      • (Duration of test recording = End Time of that time slot)
    • ► Check audio feed from Daley LL105
      • Daley LL105 theater does not have celling microphones. Audio feedback can only be received using the clip-on wireless microphone used by instructor
      • To check the audio feedback user can hover the mouse pointer to the LL105 class on Classroom Video Screen.
      • If you do not hear the instructor even after they start lecturing, they may not be using the wireless mic or may have forgotten to switch it on. To remind them, you can call the calling the SCA A/V support office in Daley lower Level at extension 2-1297 and request them to go and remind the instructor about the wireless mic.
    • ► Check audio feed from every classroom
      • To check the audio feedback, hover the mouse pointer to the room on the Classroom Video Screen.
      • Repeat the audio check every 15-20 minute throughout the duration of the recordings.
    • ► Continue monitoring the recording statuses and audio for all classrooms throughout the duration of the classes. If any issues are observed or reported, follow up immediately.
