Classroom Message Admin Message

Instructors have the option of sending a support request to the COL team via a chat option that is built into the COL Agent application on the classroom IPC (Windows only) They do this by right clicking on COL Agent icon on the taskbar, and selecting the option “Instructor Remote Assistance”, which opens a chat window.

The message sent by the instructor is received by all the currently logged in RA admins, with an alert on their RA profile. On Admin side alert message pop picture is below. Any RA admin can then respond to the message and have a conversation with the instructor. Note that these chat conversations can only be initiated by instructor from the classroom.
