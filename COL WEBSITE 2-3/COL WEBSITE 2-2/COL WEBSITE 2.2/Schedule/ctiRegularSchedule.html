CTI Regular Schedule Temporary Schedule Insert Temp Schedule

Displays the regular/recurring schedules of all the class lectures to be recorded in a particular quarter. There is a search bar where one can filter the scheduled recordings by Quarter, Weekday, ClassroomGrp, ClassName. There is also a “Send Mail” feature using which the instructors of the selected classes can be emailed (via the default email client). Below the search bar, the Begin and End Dates of the current ongoing quarter are displayed

]
  • ► Quarter: Classes can be searched quarter wise. Default = Current Quarter.
  • ► Weekday: Classes can be searched using a particular day of the week. Default = is ALL
  • ► ClassroomGrp: Search by RA classroom group. Default = ALL.
  • ► ClassName: Classes can be searched by all or part of a Course ID.
    • CSC 555-501 returns all recordings of the course CSC 555-501
    • 555 returns all recordings of all 555 courses, ANI 555, CSC 555, TDC 555 etc.
    • -501 returns all recordings of all courses with section 501

❖ If scheduled classes are searched with default values, it looks like this.

❖ CTI Scheduled Class Result: Displays future classes to be recorded with information (i.e.).

  • ► Class Name: Displays the class name with class ID.
  • ► Classroom: Displays the Room information where class is to be recorded with Building Name and Room Number
  • ► Weekday: Displays the Day of the week on which class to be recorded
  • ► Start Time: Displays the start-time of the lecture
  • ► Instructor: Displays Instructor Name of the particular class.
  • ► Length: Displays the recording duration. E.g. 1:35:00 means 1 hour 35 min recording.
  • ► File Server: Displays the server information on which it is to be uploaded
  • ► Email: Displays email address of the Instructor.
